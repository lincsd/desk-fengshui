第1章：5个让你越来越不顺的工位大忌

先排雷，再布局。很多人不是没有好运，而是工位一直在悄悄消耗状态。

1. 桌面过乱

表现文件堆积、零食散落、线缆缠绕、杯子太多。

影响视觉噪音高，注意力下降，领导印象分变差。

解法桌面只留“正在用”的东西，下班前 3 分钟归位。

2. 背后无靠

表现背后是走廊、开放空间、窗户。

影响容易紧绷、难专注，也容易感觉没人撑你。

解法优先靠墙；不行就高靠背椅 / 靠垫 / 椅背小葫芦。

3. 正对厕所或茶水间

表现抬头就是厕所门、茶水区、人流口。

影响噪音、气味、人流会持续打断专注力。

解法铜葫芦口朝问题方向，高绿植做遮挡，必要时加淡香型香薰。

4. 头顶压迫

表现横梁、管道、空调直吹。

影响压迫感、易疲劳、易生病。

解法换位最佳；否则用向上生长的植物 + 挡风板缓解。

5. 屏幕脏乱

表现满屏指纹、灰尘、油污。

影响眼睛疲劳、效率下降、心理上产生“前途模糊”的感觉。

解法每周一次清洁湿巾擦屏，1 分钟见效。

快速判断：如果你中了 3 条以上，先不要急着买摆件，先把问题位清掉，效果反而更快。

第2章：办公桌九宫格布局法

这是整本书最核心的方法。不会看朝向也没关系，先用“坐下后的左右前后”就够了。

┌──────────────┬──────────────┬──────────────┐
│    左后方      │    正后方      │    右后方      │
│   靠山位        │   事业位        │   净化位        │
│  放水晶/靠垫    │  放文昌塔       │  放盐灯        │
├──────────────┼──────────────┼──────────────┤
│    正左方      │    正中央      │    正右方      │
│   财位          │   工作区        │   白虎位        │
│  放招财猫/水晶  │  保持整洁       │  放虎皮兰       │
├──────────────┼──────────────┼──────────────┤
│    左前方      │    正前方      │    右前方      │
│   生气位        │   明堂          │   文案位        │
│  放绿植         │  保持空旷       │  放文竹        │
└──────────────┴──────────────┴──────────────┘

4 条核心原则

  1. 左高右低：左边宜高宜动，右边宜低宜静。
  2. 前方空旷：正前方别堆文件，明堂要留白。
  3. 后方有靠：靠墙最好，没有就补靠背与靠垫。
  4. 中央清净：主战场只留电脑和当前工作物。
实操 4 步：清空桌面 → 擦桌面 → 按九宫格摆回去 → 拍照留存，之后就按这个标准维护。

选购和摆放要点

招财猫

  • 优先陶瓷、太阳能款。
  • 面朝门口或走道，不要朝自己。
  • 放左边，不放右边。

黄水晶

  • 新手选原石，管理层选球，销售选簇。
  • 每月清水冲洗，满月可晒月光。

貔貅

  • 黄铜最稳，黑曜石适合压力大的人。
  • 别让人乱摸，别正对镜子。

文昌塔

  • 普通打工人用 7 层或 9 层足够。
  • 下面可压职业目标纸条。

盐灯

  • 办公桌买小号 USB 款即可。
  • 别放潮湿位置。

铜葫芦

  • 对厕所：口朝厕所。
  • 背后走廊：挂椅背或桌后方。
  • 空调直吹：摆在冲煞位置做缓冲。

附录：常见问题

Q1：我不信风水，有必要吗？

有。即使抛开玄学，一个更整洁、更有秩序、更有审美和心理暗示的工作环境，本身就能提升效率和心情。

Q2：同事会不会觉得奇怪？

不会。优先用低调款：绿植、陶瓷摆件、盐灯、简约香薰，都会更像“高级桌面布置”。

Q3：总预算多少合适？

基础方案 100-200 元足够；想做得完整些，300-500 元也能封顶。

Q4：经常换工位怎么办？

优先使用可移动物：手串、小摆件、便携香薰。人到哪，开运物到哪。

Q5：在家办公能不能用？

完全可以，而且家里更自由，布局空间通常比公司更大。

写在最后：桌面风水不是万能解法，但它能让你在努力之外，多一层稳定心态、优化状态、增强行动感的辅助系统。
